# Cookbook: HDFS

*Note: See the corresponding [lecture notes about HDFS](/notes/hdfs.html). This page has cookbook recipes.*

## Get help

```
$ hdfs dfs -help
```

## List files

```
$ hdfs dfs -ls /
Found 1 items
drwxrwx---   - root supergroup          0 2015-01-02 11:15 /tmp
```
